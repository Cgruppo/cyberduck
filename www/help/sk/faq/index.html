Nápoveda pre Cyberduck
Často kladené otázky

Čo je Cyberduck?
Cyberduck je „open source“ FTP a SFTP (SSH zabezpečený prenos súborov) prehliadač s licenciou GPL.

Je Cyberduck zadarmo?
Cyberduck je slobodný softvér. Slobodný softvér znamená slobodu užívateľa spúšťať, kopírovať, distribuovať a vylepšovať program. Ak je pre vás táto aplikácia užitočná, prosím pouvažujte o jej podpore. Podporou prejavíte nielen uznanie, ale súčasne podporíte aj jej ďalší vývoj. Pomôžete tak vylepšiť Cyberduck! Viď http://cyberduck.ch/donate.

Cyberduck nie je dostupný v mojom jazyku!
A prečo ho neskúsite preložiť sami? Kontaktujte ma a ja vám pošlem príslušné súbory na preklad. Aktuálne dostupné lokalizácie sú: angličtina, čeština, hebrejčina, holandština, fínština, francúzština, nemčina, taliančina, japončina, kórejčina, nórština, portugalčina, slovenčina, španielčina, čínština (tradičná a zjednodušená), ruština, švédština, maďarčina, dánčina, poľština, indonézština, katalánčina, welština a thajčina.

Aké sú systémové požiadavky?
Cyberduck funguje v Mac OS X 10.3.9 alebo novšom.

Ako môžem odoslať hlásenie o chybe?
Vytvorte nový záznam („ticket“) využitím vyhľadávača problémov s popisom chyby a postupu, ktorým ste sa k nej dopracovali. Ešte predtým skúste prezrieť alebo prehľadať už jestvujúce záznamy.

Môžem požiadať o doplnenie novej funkcie?
Najskôr prezrite alebo prehľadajte jestvujúce záznamy, či už vaša požiadavka nebola zaznamenaná skôr. Ak nie, vytvorte nový záznam („ticket“) popisujúci chýbajúcu funkciu.

Ako nastavím Cyberduck ako východziu aplikáciu pre FTP umiestnenia?
Cyberduck môžete nastaviť ako východzí manipulačný program pomocou RCDefaultApp.

Ako sprístupním záložky Cyberducku v aplikácii Quicksilver?
Otvorte predvoľby aplikácie Quicksilver a nainštalujte Cyberduck modul. Potom, pri výbratom Cyberducku v Quicksilveri, použite šípku vpravo.

Prečo Cyberduck neobsahuje prehliadač lokálnych súborov?
Na to slúži aplikácia Finder. Použite metodu „ťahaj a pusti“ (prípadne spolu s Exposé), alebo ponuku Súbor → Odoslať. Rovnako tak môžete skopírovať súbory vo Finderi a následne ich vložiť do aplikácie Cyberduck využitím ponuky Úpravy → Vložiť z Finderu.

Aké proxy servery Cyberduck podporuje?
SOCKS 4/5. Nastavenia proxy možno upraviť v systémových predvoľbách.

Kde Cyberduck uchováva svoje predvoľby?
~/Library/Application Support/Cyberduck/ a ~/Library/Preferences/ch.sudo.cyberduck.plist

Má Cyberduck nejaké skryté vlastnosti?
Niektoré vlastnosti nie sú dostupné v predvoľbách, pretože zatiaľ nie sú dostatočne stabilné, prípadne nie sú vo všeobecnosti významné. Nasledovné vlastosti môžete sprístupniť príslušnými príkazmi v aplikácii Terminal.

  • Zavrieť lištu záložiek po nadviazaní spojenia:
    defaults write ch.sudo.cyberduck browser.closeDrawer true

  • Neukladať záložky:
    defaults write ch.sudo.cyberduck favorites.save false

  • Nemeniť ikonu dokumentu na ikonu Cyberducku v priebehu prevzatia:
    defaults write ch.sudo.cyberduck queue.download.updateIcon false

  • Zavrieť okno priebehu po ukončení prenosu súborov:
    defaults write ch.sudo.cyberduck queue.orderBackOnStop true

  • Vypnúť podporu služby Bonjour:
    defaults write ch.sudo.cyberduck rendezvous.enable false

  • Vypnúť podporu služby Growl:
    defaults write ch.sudo.cyberduck growl.enable false

  • Vypnúť automatické rozbaľovanie načítaných zložiek v zobrazení „Rozšírený zoznam“:
    defaults write ch.sudo.cyberduck browser.view.autoexpand false

  • Globálne vylúčiť niektoré súbory a zložky pri preberaní a odosielaní, napríklad:
    - vylúčiť všetky CVS a .svn zložky z prevzatia:
    defaults write ch.sudo.cyberduck queue.download.skip "CVS .svn"
    - vylúčiť všetky CVS a .svn zložky z odosielania:
    defaults write ch.sudo.cyberduck queue.upload.skip "CVS .svn"
    Tieto nastavenia sa aplikujú pri preberaní a odosielaní súborov pri využití príkazu „Synchronizovať“.
