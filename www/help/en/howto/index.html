Configure SSH Public Key Authentication

  • Toolbar button.

    Click the New Connection toolbar button or File → New Connection.

  • Quick Connect.

    Type in the name of the server directly in to the Quick Connect field in the toolbar. The text field will autocomplete from bookmarked hosts. You can enter a string in the format user@host, i.e. user@example.net, or a fully qualified URL such as ftp://mirror.switch.ch/mirror/.

  • From bookmarks.

    Select View → Toggle Bookmark Drawer and double click a bookmark. You can also access the bookmarks from the Bookmark menu.

  • From the history.

    Select a bookmark from the menu Bookmark → History.

  • From Bonjour discovered services.

    Servers in the local network are most likely to appear in the menu Bookmark → Bonjour.

  • From the Finder.app.

    Double click a Cyberduck bookmark file in the Finder.

    Double click a Internet Location File in the Finder.app (requires RCDefaultApp from http://www.rubicode.com/Software/RCDefaultApp/)
