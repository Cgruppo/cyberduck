Cyberduck Help
Common Problems

Special characters such as Umlaute aren't displayed correctly in the browser.
Try to change the character encoding used. See 'View' → 'Text Encoding'. Try UTF-8 (the default), ISO-8859-1 and Windows-1252.

The folder size isn't displayed correctly.
This is not the size of its content but the size of the folder itself. Use 'File' → 'Info' to calculate the size recursively of all contained files.

It is not possible to change permissions. The error message FTP Error: SITE not understood or similar is displayed.
The server does not support this feature (which is an optional extension to the FTP protocol) and can not be used.

The connection to the server frequently times out; I want it to stay it alive if at all possible!
You can force Cyberduck to send a 'no operation' command at a given time interval. See the 'Advanced' tab in the Preferences.

I often get an error titled IO Error: Broken pipe.
There was a problem with the unerlying network stream either because the connection was too slow or the server wouldn't give a response.

I get an error titled IO Error: Read Timed Out when attempting to open a connection.
There is either a firewall blocking requests or no server is listening at the given address and protocol.

I can't select any external editor.
Install one of the supported editors and launch it first. Also relaunch Cyberduck if the editor is still not selectable. If that still doesn't work out, please refer to this article.
