Cyberduck Help
FAQ

What is Cyberduck?
Cyberduck is a open source FTP and SFTP (SSH Secure File Transfer) browser licenced under the GPL.

Is Cyberduck free?
Cyberduck is free software. Free software is a matter of the users' freedom to run, copy, distribute, study, change and improve the software. If you find this program useful, please consider making a donation. A donation would not only demonstrate your appreciation of this software, but also help to advance development in the future. It will help to make Cyberduck even better! Please see http://cyberduck.ch/donate.

Cyberduck isn't available in my language!
Why not translate it yourself? Contact me and I'll send you the relevant files. Current available localizations are English, Czech, Dutch, Finnish, French, German, Italian, Japanese, Korean, Norwegian, Portuguese, Slovak, Spanish, Chinese (Traditional and Simplified), Russian, Swedish, Hungarian, Danish, Polish, Indonesian, Catalan, Welsh and Thai.

What are the requirements to run Cyberduck?
Cyberduck runs on Mac OS X 10.3.9 or later.

How do I submit a bug report?
Open a new ticket using the issue tracker with a description of what you have a done and what went wrong. Make sure to look or search for existing tickets first.

How do I request a feature?
Look or search first in the issue tracker if this hasn't been requested before. If not, then open a new ticket describing the missing functionality.

How can I make Cyberduck the default application for FTP URLs?
You can define Cyberduck as the default protocol handler with RCDefaultApp.

How can I access Cyberduck bookmarks with Quicksilver?
Open the Quicksilver Preferences and install the Cyberduck Module. Then when Cyberduck is selected in Quicksilver use the right-arrow key.

Why isn't there a local file viewer in Cyberduck?
That's what Finder.app is for. Use drag and drop (eventually together with Exposé) or Menu File → Upload instead You can also copy files in the Finder.app and then paste them in Cyberduck using Menu Edit → Paste from Finder.

What proxy servers does Cyberduck support?
SOCKS 4/5. You can adjust the proxy settings in the System Preferences.

Where does Cyberduck store its preferences?
~/Library/Application Support/Cyberduck/ and ~/Library/Preferences/ch.sudo.cyberduck.plist

Are there any unknown features of Cyberduck?
There are some settings which aren't yet available in the Preferences either because they are not considered stable yet or not of general interest. These are the following: (To enable one of these features type the commands in a Terminal.app window.)

  • Close bookmark drawer when opening connection
    defaults write ch.sudo.cyberduck browser.closeDrawer true

  • Don't save bookmarks
    defaults write ch.sudo.cyberduck favorites.save false

  • Don't change document icon to Cyberduck icon while downloading
    defaults write ch.sudo.cyberduck queue.download.updateIcon false

  • Close transfer window uppon completed transfer
    defaults write ch.sudo.cyberduck queue.orderBackOnStop true

  • Disable Bonjour support
    defaults write ch.sudo.cyberduck rendezvous.enable false

  • Disable Growl support
    defaults write ch.sudo.cyberduck growl.enable false

  • No autoexpanding (spring loaded folders) in outline view
    defaults write ch.sudo.cyberduck browser.view.autoexpand false

  • Exclude some files and directories globally from download, upload, e.g.
    - Exclude all CVS and .svn directories from downloads:
    defaults write ch.sudo.cyberduck queue.download.skip "CVS .svn"
    - Exclude all CVS and .svn directories from uploads:
    defaults write ch.sudo.cyberduck queue.upload.skip "CVS .svn"
    These settings also apply to downloaded or uploaded using the synchronization feature.
