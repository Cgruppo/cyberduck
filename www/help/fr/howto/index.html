Configurer l'authentification par clé publique SSH

  • Bouton de la barre d'outils.

    Cliquez sur le bouton de la barre d'outils Nouvelle connexion ou Fichier → Se connecter à…

  • Connexion rapide.

    Tapez le nom du serveur directement dans le champ Connexion rapide de la barre d'outils. Le champ texte se remplira automatiquement selon les serveurs enregistrés dans les signets. Vous pouvez saisir une chaîne sous la forme utilisateur@serveur, p. ex. utilisateur@exemple.net, ou une URL adaptée telle que ftp://mirror.switch.ch/mirror/.

  • À partir des signets.

    Choisissez Présentation → Afficher/Masquer les Signets et double cliquez sur un signet du tiroir des Signets. Vous pouvez aussi accéder aux signets par le menu Signet.

  • À partir de l'historique.

    Choisissez un signet par le menu Signet → Historique.

  • À partir des services Bonjour disponibles.

    Des serveurs du réseau local sont susceptibles d'être présents dans le menu Signet → Bonjour.

  • À partir du Finder.

    Double-cliquez sur un fichier signet de Cyberduck dans le Finder.

    Double-cliquez sur un fichier Adresse Internet du Web dans le Finder (nécessite RCDefaultApp de http://www.rubicode.com/Software/RCDefaultApp/)
