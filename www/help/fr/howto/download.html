Configurer l'authentification par clé publique SSH

  • Double-clic.

    Double-cliquez sur un fichier dans la fenêtre du navigateur pour le télécharger vers le dossier par défaut indiqué dans Cyberduck → Préférences → Transferts → Télécharger vers.

  • Télécharger par glisser-déposer.

    Glissez-Déposez les fichiers ou les dossiers à l'endroit où vous voulez les télécharger dans le Finder.

  • Bouton de la barre d'outlis.

    Personnalisez la barre d'outil du navigateur par le menu Présentation → Personnaliser la barre d'outils… pour ajouter le bouton Télécharger à la configuration par défaut.

  • Le menu de téléchargement.

    Choisissez Fichier → Télécharger ou faites un clic droit sur le fichier dans le navigateur et sélectionnez Télécharger dans le menu contextuel.

  • Comment mettre un fichier en attente pour le télécharger plus tard ?

    Glissez le fichier de la fenêtre du navigateur vers celle des Transferts en attente.
