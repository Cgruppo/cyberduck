  • Comment éditer un fichier avec un éditeur externe ?

    Vous pouvez modifier un fichier distant comme un fichier local en cliquant sur le bouton Édition dans la barre d'outils ou en choisissant Fichier → Édition. Le fichier sera téléchargé dans le dossier temporaire de l'utilisateur et ouvert dans l'éditeur préféré (Les éditeurs compatibles sont SubEthaEdit, BBEdit, TextWrangler, TextMate, Text-Edit Plus et d'autres). Choisissez votre éditeur préféré dans Cyberduck → Préférences. Le fichier sera transféré au serveur chaque fois que choisirez Fichier → Enregistrer (ou File → Save) dans le programme d'édition. Le fichier ne subit pas de modification si vous le fermez sans enregistrer.

  • Comment ne changer que le nom du fichier ?

    Sélectionnez le fichier dans le navigateur et pressez la touche Entrée, ou Retour. Tapez le nouveau nom et pressez la touche Retour à nouveau pour sortir du mode d'édition. Vous pouvez aussi renommer un fichier en choisissant Fichier → Lire les informations ou en pressant le bouton Lire les informations de la barre d'outil. Entrez simplement le nouveau nom dans le champ supérieur. Le champ doit perdre le focus (p. ex. en pressant Entrée ou Tab) pour valider le changement de nom.

  • Comment changer les permissions sur un fichier en particulier ?

    Sélectionnez le fichier dans le navigateur et choisissez Fichier → Lire les informations. Vous pouvez aussi sélectionner plusieurs fichiers à la fois. N'oubliez pas de cliquer sur le bouton Appliquer pour valider les modifications.

  • Puis-je déplacer des fichiers sur un serveur distant ?

    Vous pouvez déplacer des fichiers dans le navigateur comme vous le feriez dans le Finder.
