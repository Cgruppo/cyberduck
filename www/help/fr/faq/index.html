Aide de Cyberduck
Foire Aux Questions

Cyberduck, qu'est-ce que c'est ?
Cyberduck est un vavigateur FTP et SFTP (SSH Secure File Transfer [Transfert Sécurisé de Fichiers]) open source sous licence GPL.

Est-ce que Cyberduck est gratuit ?
Cyberduck est un logiciel libre. L'expression “logiciel libre” fait référence à la liberté pour les utilisateurs d'exécuter, de copier, de distribuer, d'étudier, de modifier et d'améliorer le logiciel. Si vous trouvez ce programme utile, merci de penser à faire un don. Un don ne démontrera pas seulement que vous appréciez ce logiciel, mais aidera aussi à son développement futur. Cela aidera à le rendre meilleur encore ! Merci de consulter http://cyberduck.ch/donate.

Cyberduck n'est pas disponible dans ma langue !
Pourquoi ne le traduisez-vous pas vous-même ? Contactez-moi et je vous enverrai les fichiers concernés. Actuellement, Cyberduck est disponible en anglais, tchèque, hollandais, finnois, français, allemand, italien, japonais, coréen, norvégien, portugais, slovaque, espagnol, chinois (traditionel et simplifié), russe, suédois, hongrois, danois, polonais, indonesien, catalan, galois et thaïlandais.

Quelle est la configuration requise pour faire fonctionner Cyberduck?
Cyberduck tourne sous Mac OS X 10.3.9 ou plus récent.

Comment puis-je rappporter un bug ?
Créez un nouveau ticket en vous aidant du traqueur de problèmes. Décrivez ce que vous étiez en train de faire et ce qui s'est passé. Pensez d'abord à regarder ou à rechercher s'il n'existe pas déjà de tickets sur le même sujet.

Comment dois-je faire pour demander une nouvelle fonctionnalité ?
Regardez ou recherchez d'abord dans le traqueur de problèmes si elle n'a pas déjà été demandée. Si ce n'est pas le cas, alors créez un nouveau ticket décrivant la fonctionnalité qui vous manque.

Comment faire de Cyberduck l'application par défaut pour les adresses FTP ?
Vous pouvez définir Cyberduck en tant que gestionnaire par défaut de ce protocole avec RCDefaultApp.

Comment puis-je accéder aux signets de Cyberduck avec Quicksilver ?
Ouvrez les Préférences de Quicksilver et installez le  plug-in pour Cyberduck. Puis, quand Cyberduck est sélectionné dans Quicksilver, utilisez la touche flèche-droite.

Pourquoi n'y a-t-il pas de visionneuse des fichiers locaux dans Cyberduck ?
C'est à ça que sert le Finder. Utilisez le Glisser-Déposer (éventuellement avec Exposé) ou alors faites Fichier → Transférer. Vous pouvez aussi copier les fichiers dans le Finder pour ensuite les coller dans Cyberduck à l'aide de Edition → Copier depuis le Finder.

Quels serveurs proxys Cyberduck accepte-t-il ?
SOCKS 4/5. Vous pouvez configurer les proxys dans les Préférences Système.

Où Cyberduck conserve-t-il ses préférences?
~/Bibliothèque/Application Support/Cyberduck/ et ~/Bibliothèque/Preferences/ch.sudo.cyberduck.plist

Existe-t-il des caractéristiques inconnues de Cyberduck?
Il existe quelques réglages qui ne sont pas encore disponibles dans les Préférences soit parce qu'ils sont considérés comme pas encore assez stables soit parce qu'ils ne sont pas d'un intérêt général. On trouve les suivants : (Pour activer ces options, tapez les commandes dans une fenêtre du Terminal.)

  • Ferme le tiroir des Signets à l'ouverture d'une connexion
    defaults write ch.sudo.cyberduck browser.closeDrawer true
  • N'enregistre pas les signets
    defaults write ch.sudo.cyberduck favorites.save false
  • Ne remplace pas l'icône du document par celle de Cyberduck lors d'un téléchargement
    defaults write ch.sudo.cyberduck queue.download.updateIcon false
  • Désactive le support de Bonjour
    defaults write ch.sudo.cyberduck rendezvous.enable false
  • Désactive le support de Growl
    defaults write ch.sudo.cyberduck growl.enable false
