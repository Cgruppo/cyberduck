FAQ

Cyberduck とは何ですか？
Cyberduck は、オープンソースの FTP／SFTP（SSH Secure File Transfer）ブラウザです。ライセンス形態は GPL です。

Cyberduck はフリー（無料）ですか？
Cyberduck はフリーソフトウェアです。ここで言うフリーとは、ユーザがソフトウエアを実行・コピー・配布・研究・変更および改良する自由のことを意味します。当プログラムをお気に入りいただけたなら、ぜひとも寄付をご検討ください。将来にわたり開発を進めていくための手助けとなります。つまり、Cyberduck をより良くする原動力となるのです！詳細は http://cyberduck.ch/donate をご参照ください。

Cyberduck が私の国の言語に対応していません！
ぜひ翻訳してください。作者にご連絡いただければ関連ファイルをお送りします。現状で対応済みの言語は以下のとおりです： 英語、チェコ語、オランダ語、フィンランド語、フランス語、ドイツ語、イタリア語、日本語、ハングル語、ノルウェー語、ポルトガル語、スロバキア語、スペイン語、中国語（繁体／簡体）、ロシア語、スウェーデン語、ハンガリー語、デンマーク語、ポーランド語、インドネシア語、カタロニア語、ウェールズ語、タイ語。

Cyberduck の動作条件は？
Cyberduck は Mac OS X 10.3.9 以降で動作します。

不具合を報告するにはどうしたらよいでしょう？
issue tracker で new ticket を開き、どのような操作でどのような問題が発生したかを記述してください。まずは既存の ticket を見るまたは検索してください。

機能の追加をリクエストするには？
まずは issue tracker 内を見るまたは検索し、同じ機能がすでにリクエストされていないかご確認ください。まだのようなら、new ticket を開いて追加したい機能の内容を記述してください。

Quicksilver で Cyberduck のブックマークにアクセスするには？
Quicksilver Preferences を開き Cyberduck Module をインストールします。Quicksilver で Cyberduck を選択し、右矢印キーを押してください。

なぜ Cyberduck にはローカルファイルビューアがないのですか？
Finder はそのためにあるのです。ドラッグ＆ドロップ操作（場合により Exposé と併用）か、または‘ファイル’→‘アップロード’を選択してください。Finder 上でコピーしたファイルを、Cyberduck にペーストすることもできます。‘編集’→‘Finder からペースト’を選択してください。

Cyberduck はどのプロキシサーバに対応していますか？
SOCKS 4/5 です。プロキシの設定はシステム環境設定で変更できます。

Cyberduck の設定ファイルはどこに保存されますか？
~/Library/Application Support/Cyberduck/ および ~/Library/Preferences/ch.sudo.cyberduck.plist です。

Cyberduck には隠し機能はありますか？
環境設定から選択できない設定には、以下のようなものがあります。これらは現状ではまだ不安定か、またはあまり需要がないと思われるものです。これらを使用するには、ターミナルからコマンドを入力してください：

  • 接続する際にブックマークパネルを閉じる
    defaults write ch.sudo.cyberduck browser.closeDrawer true
  • ブックマークを保存不可にする
    defaults write ch.sudo.cyberduck favorites.save false
  • ダウンロード中のファイルのアイコンを Cyberduck のアイコンにしない
    defaults write ch.sudo.cyberduck queue.download.updateIcon false
  • Bonjour サポートを停止する
    defaults write ch.sudo.cyberduck rendezvous.enable false
  • Growl サポートを停止する
    defaults write ch.sudo.cyberduck growl.enable false

愛用のエディタが未対応なのですが。
外部エディタプロトコル（ODB）に対応したエディタが必要です。現状では任意のエディタを使用することはできません。
