Cyberduck ヘルプ
使い方

  • 接続開始
  • ブックマーク
  • ダウンロード
  • アップロード
  • ファイルの編集
  • SSH
  • 環境設定

  • Dashboard ウィジェット

  • ウィジェットを開く。

    Dashboard アイコンをクリックするか、または F12 キーを押します。‘+’アイコンをクリックしてウィジェットバーを表示させ、‘Cyberduck’をクリックします。

  • 最初の設定。

    ウィジェットを開いたら、次は設定です。ブルーの四角いウインドウ上に‘設定してください！’というメッセージが表示されます。このメッセージをクリックして、ウィジェットを裏返してください。

  • 接続の設定。

    プルダウンメニューからブックマークを選択してください；ブックマークは Cyberduck アプリケーションと共有されますので、日頃お使いの接続先が見つかるはずです。各項目に変更を加えても、ウィジェット上にしか影響はありません。

  • アップロード。

    いったん Dashboard を隠します。Finder や iPhoto ライブラリなどからファイルを選択し、ドラッグを開始してマウスボタンを押したまま Dashboard を表示します。ファイルをウィジェットの上までドラッグし、カーソルがグリーンの‘+’に変わったらドロップしてください。
