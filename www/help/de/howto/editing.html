  • Wie kann ich eine Datei in einem externen Editor bearbeiten?

    Sie können eine Datei genau wie eine lokale Datei bearbeiten, indem Sie in der Symbolleiste auf Editieren klicken oder den Menüeintrag Ablage → Bearbeiten wählen. Die Datei wird dann in das temporäre Verzeichnis des Benutzers geladen und mit dem voreingestellten Editor geöffnet (unterstütze Editoren sind u.a. SubEthaEdit, BBEdit, TextWrangler, TextMate, Text-Edit Plus). Unter Cyberduck → Preferences können Sie Ihren gewünschten Editor festlegen. Immer wenn Sie die Datei im Editor sichern, wird diese zum Server hochgeladen. Die Datei auf dem Server wird nicht verändert, wenn Sie den Editor ohne zu sichern schließen.

  • Wie ändere ich den Dateinamen?

    Wählen Sie die Datei im Browserfenster aus und drücken Sie die Return- bzw. Enter-Taste. Tippen Sie den neuen Namen ein uns schließen Sie Ihre Eingabe mit Return/Enter ab, um den neuen Namen festzulegen. Sie können Dateien auch mit Ablage → Info oder über den Info-Knopf in der Symbolleiste umbenennen. Geben Sie den neuen Namen einfach im obersten Feld ein und schließen Sie die Eingabe durch drücken der Enter-/Return- oder der Tab-Taste ab, um die Änderung zu übernehmen.

  • Wie ändere ich die Zugriffsrechte bei einer bestimmten Datei?

    Wählen Sie die Datei im Browserfenster aus und wählen dann den Menüeintrag Ablage → Info. Sie können auch mehrere Dateien gleichzeitig auswählen. Denken Sie nach dem Ändern daran, den Anwenden-Knopf zu drücken, um die Änderungen zu übernehmen.

  • Kann ich die Dateien auf dem Server verschieben?

    Sie können die Dateien auf dem Server im Browserfenster genauso verschieben wie im Finder.
