  • Dateien können synchronisiert werden, indem sie das zu synchronisierende Verzeichnis im Browserfenster auswählen und den Menüeintrag Ablage → Synchronisieren auswählen.

    Sie werden anschließend nach dem Verzeichnis auf Ihrem Computer gefragt, das mit dem auf dem Server abgeglichen werden soll. Cyberduck versucht nur, Dateien unterschiedlicher Größen zu synchronisieren. Abhängig vom Änderungsdatum wird die Datei auf den Server hoch- oder vom Server heruntergeladen. Zuvor wird Ihre Bestätigung hierzu und ob fehlende Dateien hoch/heruntergeladen oder gespiegelt werden sollen eingefordert..
