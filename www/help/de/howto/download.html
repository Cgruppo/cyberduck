  • Doppelklick.

    Doppelklicken Sie eine Datei im Browserfenster, um diese in das Standard-Downloadverzeichnis (festgelegt unter Cyberduck → Einstellungen → Dateitransfers) herunterzuladen.

  • Download mit Drag-and-Drop..

    Ziehen Sie die Dateien einfach in das gewünschte Verzeichnis im Finder.

  • Download-Knopf in der Symbolleiste.

    Sie können den Download-Knopf der Symbolleiste hinzufügen, indem Sie Darstellung → Symbolleiste anpassen wählen und ihn in die Standard-Konfiguration aufnehmen.

  • Der Download-Menüeintrag.

    Wählen Sie Ablage → Download oder wählen Sie im Kontextmenü (Rechtsklick oder ctrl-Klick) der herunterzuladenden Datei den Eintrag Download.

  • Wie kann ich eine Datei der Download-Warteschlange hinzufügen?

    Ziehen Sie die betreffende Datei vom Browser- in das Dateitransfers-Fenster.
