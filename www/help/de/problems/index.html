Cyberduck Hilfe
Bekannte Probleme

Spezielle Zeichen (wie z.B. Umlaute) werden im Browser nicht korrekt dargestellt.
Versuchen Sie, die Zeichenkodierung zu ändern. Befehl Darstellung → Zeichenkodierung. Probieren Sie UTF-8 (Standard), ISO-8859-1 und Windows-1252 aus.

Die Ordnergröße wird nicht richtig angezeigt.
Die angezeigte Größe bezieht sich nicht auf die Inhalte des Ordners, sondern auf den Ordner selbst.

Ich kann die Zugriffsrechte nicht verändern. Es erscheint die Meldung FTP Error: SITE not understood (o.ä.).
Der Server unterstützt diese Funktion nicht (die übrigens eine optionale Funktion im FTP-Protokoll ist) und kann nicht benutzt werden.

Die Wartezeit beim Verbindungsaufbau wird häufig überschritten; ich will - falls möglich - länger versuchen, eine Verbindung aufzubauen!
Sie können Cyberduck anweisen, in gewissen Abständen ein "no operation"-Kommando ("NOOP") zu senden (Einstellungen → Erweitert)

Ich erhalte oft die Fehlermeldung " IO Error: Broken pipe ".
Es liegt vermnutlich ein Problem mit der Netzwerkverbindung vor, da diese entweder zu langsam ist oder der Server nicht geantwortet hat.

Ich erhalte die Fehlermeldung " IO Error: Read Timed Out " beim Versuch , eine Verbindung aufzubauen.
Entweder blockiert eine Firewall die Anfragen oder kein Server erwidert sie unter der angegebenen Adresse bzw. dem angegebenen Protokoll.

Ich kann keinen externen Editor festlegen.
Installieren Sie einen der unterstützten Editoren und starten Sie diesen zuerst. Wenn sich der Editor immernoch nicht auswählen lässt, starten Sie Cyberduck neu. Sollte auch dies nicht funktionieren, lesen Sie bitte diesen Artikel (auf Englisch).

Mein Lieblingseditor wird nicht unterstützt.
Ein Editor muss das "External Editor Protokoll" (ODB) unterstützen.
