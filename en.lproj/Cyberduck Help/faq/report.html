Where does Cyberduck store its preferences?

Write an email to bugs@cyberduck.ch with a description what you have a done and what went wrong. It would help if you can include the crash report and also all the files with the pattern ~/Library/Logs/JavaNativeCrash_pid(****).log. Also please copy the output of the console.log available using the Console.app from /Applications/Utilities/Console.app.
