Cyberduck runs on Mac OS X 10.3.9 or later and needs Java Runtime which is bundled with Mac OS X. If Cyberduck crashes when launching, try installing the latest Java Update from http://www.apple.com/support/downloads/javaupdate142.html or if you are running OS X 10.4 make sure to install the latest Java Update available through Software Update.
