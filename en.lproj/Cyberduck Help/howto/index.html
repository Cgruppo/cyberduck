Click the 'New Connection' button in the toolbar or the 'File' Menu. You must have a browser window (command-N) open to do this. Fill in the relevant information and/or choose a server from the 'Bonjour' or 'Bookmarks' drop down menu.

Type in the name of the server directly in to the 'Quick Connect' field in the toolbar. The text field will autocomplete from bookmarked hosts. You can enter a string in the format user@host, i.e. user@example.net, or a fully qualified URL such as ftp://mirror.switch.ch/mirror/.

Use the Bookmark drawer. Select 'Toggle Bookmark Drawer (command-B)' from the 'View' menu and double click an entry in the table.

Use the bookmark or history menu. Select a bookmark from the 'Bookmark' menu.

Double click a Cyberduck bookmark file in the Finder.

Double click a Internet Location File in the Finder (requires RCDefaultApp from http://www.rubicode.com/Software/RCDefaultApp/)

Does Cyberduck support Rendezvous/Bonjour/Zeroconf?

In the 'New Connection' Dialog look at the Bonjour icon in the top left corner. You will get a list with FTP/SFTP servers supporting Bonjour on your local network.
You can also see available Bonjour services in the 'Bookmark' menu.
